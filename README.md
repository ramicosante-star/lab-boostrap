# 🚀 Spazio di Rossella - Exercise Project

Benvenuti nel repository del mio primo progetto web strutturato! Questo sito è un'esercitazione pratica focalizzata sull'utilizzo di **Bootstrap 5** e sull'integrazione di flussi di lavoro professionali tramite **GitHub Actions**.

## 🛠️ Tecnologie Utilizzate fino ad oggi
* **HTML5** e **CSS3** per la struttura.
* **Bootstrap 5.3** per il design responsive e i componenti interattivi.
* **GitHub Actions** per il deploy automatico.
* **RealFaviconGenerator** per il supporto multi-piattaforma.

## ✨ Cosa sto imparando con questo progetto
- [x] Gestione dei componenti Bootstrap (Accordion, Cards, Tables).
- [x] Risoluzione di errori lato client tramite **DevTools**.
- [x] Configurazione di **Favicons** e **OpenGraph** (Work in progress).
- [x] Utilizzo professionale di **Git** (Branch, Commit, Issues).
